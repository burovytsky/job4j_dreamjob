## Project "DreamJob"

In that project has realized example of simple web application about job.
Used technologies:
- Apache Tomcat
- PostgresSQL
- Liquibase library for DB
- JS,AJAX,JSP (front)
- Logging system Slf4j with log4j
- Mockito with PowerMock for test cases
- Maven as a build system
